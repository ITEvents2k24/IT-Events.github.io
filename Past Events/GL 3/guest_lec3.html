  • Participants will learn to create a simple yet effective website using foundational technologies:
    • HTML: Structure your content and elements.
    • CSS: Style your website for a polished look.
    • JavaScript: Add interactivity and enhance user experience.

    • Creating a GitHub account and understanding its interface.
    • Setting up a new repository for your project.
    • Pushing your local files to GitHub using Git commands.
    • Enabling GitHub Pages for live hosting and sharing your work with others.
  • By the end of the workshop, attendees will have:
    • A solid understanding of the web development process from start to finish.
    • The tools necessary to build and launch their own websites independently.
    • Confidence in applying basic coding concepts to future projects.
  • Participants will receive resources and materials for continued learning after the session.
  • This workshop provides essential skills for beginners, preparing them for future web development endeavors.
